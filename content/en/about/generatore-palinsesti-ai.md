## The Project

Open source tool to automate editorial planning. Combines a fast CLI for daily use and a web interface for calendar visualization and management.

## Stack

- Python + Typer for the CLI
- FastAPI for the backend
- Local LLMs via Ollama (no API costs)
- Minimal vanilla JS interface

## Features

Automatically generates content ideas, schedules them based on desired frequency, and exports in CSV or Notion-ready format.
