## Il Progetto

Tool open source per automatizzare la pianificazione editoriale. Combina una CLI veloce per uso quotidiano e un'interfaccia web per la visualizzazione e gestione del calendario.

## Stack

- Python + Typer per la CLI
- FastAPI per il backend
- LLM locali via Ollama (nessun costo API)
- Interfaccia minimale in vanilla JS

## Funzionalità

Genera automaticamente idee di contenuto, le schedula in base alla frequenza desiderata, e produce un export in formato CSV o Notion-ready.
